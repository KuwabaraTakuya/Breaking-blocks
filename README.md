# Block Breaker / ブロック崩し

HTML と JavaScript のみで作られたブロック崩しゲームです。外部ライブラリは一切使用していません。

## プレイ方法

`index.html` をブラウザで開くだけで遊べます。

## ゲームの流れ

```
タイトル画面 → ゲーム → ゲームオーバー → タイトル画面
```

1. タイトル画面で **START** ボタンを押す
2. マウスまたはタッチ操作でパドルを左右に動かす
3. ボールでブロックを崩してスコアを稼ぐ
4. ゲームオーバーになるとスコアが表示され、タイトルへ戻れる

## ゲームオーバーの条件

- ボールがパドルに当たらずに画面下へ落ちた場合
- ブロックが画面中央付近のデンジャーラインに到達した場合

## 特徴

- **ブロック追加** — 7秒ごとに新しいブロックの列が上から出現し、既存のブロックが下へシフトする
- **パドル反射** — パドルに当たった位置によってボールの角度が変わる
- **スコア** — ブロック1個につき10点
- **ベストスコア** — セッション中のハイスコアを記録
- **次の列カウントダウン** — 画面下部のバーで次のブロック追加タイミングを表示
- **レスポンシブ対応** — PC・スマートフォン両対応（マウス／タッチ操作）
- **ネオンデザイン** — 暗背景にグロウエフェクトのビジュアル

## 操作方法

| デバイス | 操作 |
|----------|------|
| PC | マウスを左右に動かす |
| スマートフォン | 画面をタッチしてスライド |
